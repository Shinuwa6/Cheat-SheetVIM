Bienvenue sur ce CheatSheetTERM

Commande Terminal

ls Lister le contenu d'un répertoire
ls -all Lister le contenu d'un répertoire avec les fichiers cachés
ls -lh Lister le contenu d'un répertoire avec la taille en octet, ko, mo et go
cd "nom du rep" Se déplacer dans les répertoires
rm "nom_du_fichier" Supprimer un fichier
mkdir "nom_du_rep" Créer un répertoire
rmdir "nom_du_repgit" Supprimer un repertoire
pwd Connaître l'endroit où vous êtes
cp "doc_source doc_destination" Copier un document
cp -R "rep_source rep_destination" Copier un répertoires
mv "doc_source doc_destination" Déplacer un document
$ top Liste des processus en cours
$ who Pour savoir qui est connecté
$ w Pour avoir plus de détails concernant les utilisateurs
$ users Pour savoir qui est connecté en local
$ write "nom_d'utilisateur" Pour écrire à un utilisateur
$ uptime Pour savoir depuis combien de temps l'ordinateur (ou le serveur) est booté.
